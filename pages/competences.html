Php

Langage de programmation WEB

J'ai découvert le PHP pour la première fois lors de mon stage en première année chez BC Lighting. Pendant ce stage, j'ai eu l'occasion de créer une sorte de "bac à sable" pour WordPress. J'ai développé une interface graphique permettant de générer un site WordPress avec plusieurs paramètres personnalisables. Une fois créé, ce site s'ajoutait automatiquement à une liste et devenait facilement gérable depuis le menu (un environnement de test WordPress en somme). Depuis cette expérience, J'ai ensuite travaillé sur des projets en PHP pendant mon BTS.

JavaScript

J'ai continué à travailler avec JavaScript tout au long de mon BTS, notamment dans le domaine du développement web, également avec Node Deno et pReact dans le cadre de ma veille technologique et projets personnels.
